## 1.1 What is Python?

Python is ‘an interpreted, high-level, general-purpose programming language’. So – what does that
mean?

- A programming language is a language in which sets of instructions are written to perform
a given task. The instructions, known as computer code, must follow a specific set of rules.
The code is run in order to ‘do’ something, like calculate something or plot a graph.
- In computing, a general-purpose programming language is a language that is used for a
wide variety of applications.
- A high-level language is one in which the computer code is written in a way that is designed
to be easy to understand by humans (opposed to a low-level language, which is written in a
way that is much closer to how a computer works).
- An interpreted language is one in which each line of code is executed directly when the code
is run (opposed to a compiled language, in which the code is typically converted to machine
code which can then be run).

In this module, we will look at the uses of Python in scientific computing. There will be many
times during your Physics degree in which writing programs using Python will be either useful or
essential. As Physicists, we usually attempt to understand a given physical problem by writing
down mathematical equations. It may be possible to solve those equations analytically, i.e. with
a pen and paper (by rearranging them, integrating / differentiating etc). However, especially as
you progress to your higher years, you will find there are situations where you can write down
equations to describe a problem, but you can’t solve the equations mathematically. In these cases,
there are usually ways of solving the equations numerically – we will not end up with a formula
giving the result in terms of symbols, but we can calculate what the result will be for any given
values for those symbols. Even if equations can be solved analytically, Python is very helpful as
it can easily plot graphs of your results. Python can solve many more complicated problems than
any hand-held calculator can. It has a huge range of capabilities, either installed as standard or
available by installing extra packages. Another reason why we use Python is that it can be used
to control experimental apparatus. This is called interfacing. Those of you taking the Physics
laboratory in your second year will come across this.
Although there are other programming languages that we could teach, we have chosen Python
because is a good language for introducing programming. Python encourages users to write code
that is clear and easy to read, which is not always the case with other languages. It is also extremely
popular. You can easily convert your knowledge of coding to other languages later if required. Also,
Python is supported in all major operating systems, and it is open-source. This means you can use
it for free both while you are a student and after you graduate.
